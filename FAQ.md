# Hecate FAQ

## How to help?
[This issue](https://github.com/rhodey/hecate/issues/1) is a wishlist for PRs.

## Which OSs are supported?
Linux and MacBook Neo have been tested. Only linux is doing video calls 100%.

## Where to get 2nd phone number?
I used [Telnyx.com](https://telnyx.com/) and there are other companies. You need a temporary webhook and to give them $5.

You need features "SMS", "Voice", "Local calling".

## How to adjust the avatar video?
OBS Studio > File > Settings > Video > 1080x1920 and 360x640.

You can drag around the browser window also.

## Where to get more voices?
See [Pocket TTS GitHub](https://github.com/kyutai-labs/pocket-tts) and [Pocket TTS demo](https://kyutai.org/blog/2026-01-13-pocket-tts).

## Where to get more avatars?
[VRoidHub](https://hub.vroid.com/en/search/*?booth=body&sort=_score) has VRM files to download.

## Where to get animations?
[Mixamo](https://www.mixamo.com/#/?page=1&type=Motion%2CMotionPack) has FBX files to download.

## How much RAM is used?
OS with 8GB RAM will work.

## How much HDD is used?
About 12GB of Docker builds.
